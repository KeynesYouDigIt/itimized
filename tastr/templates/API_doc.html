Thanks for your interest in the tastr rest API

Please note the database is designed for development and may go offline temporarily if the site grows faster than expected.



it is currently possible to access any data on a Recipe or Ingredient in our database.

use the username to get an authors recipes

tastr.com/recipes/author/username

To get a single Recipe, the call is in the format

tastr.com/recipes/recipe id number

Ingredients are similar

tastr.com/ingredients/recipe id number



get full lists of each via

tastr.com/recipes/all

tastr.com/ingredients/all {% endblock%}
